Programme pour entrer les données d’antennes et de câbles    

Programme: HB9ZS
Traduction: HB9IJG et HB9PVC

Ce programme permet d’enregistrer des données d’antennes et de câbles personnalisées pour ensuite les utiliser dans le programme de calcul du rayonnement

On peut entrer les données pour

  • un nombre indéterminé de types d’antennes
  • jusqu’à 10 types de câbles supplémentaires

 

Lancement du programme

 

Au départ apparaît le masque pour le choix de la langue. Après sélection de la langue, le champ correspondant devient jaune. La fenêtre du programme apparaît après avoir pressé le bouton " Start ".

Affichage du masque d’entrée des antennes

Cliquez sur "Entrer les données d’antennes". Cliquez ensuite dans le menu qui apparaît sur "Masque d’entrée des antennes"

 

 

Masque d’entrée des antennes


Les champs du masque d’entrée des antennes

Les champs pour les entrées sont jaunes et se divisent en :

  • "Fabricant" : Entrez le fabricant de l’antenne.
  • "Type" : Désignation du type de l’antenne. Il permet l’identification et le stockage de ses données.
  • "Gain" : Gain de l’antenne en dBi dans la bande amateur correspondante.
  • "Atténuation angulaire" : Atténuation angulaire verticale de 0 à 90°. 0° = direction de rayonnement, 90° = verticale sous l’antenne. Comme la plupart des antennes sont orientables et que l’atténuation entre 90° et 180° n’a pas d’intérêt, on ne considèrera que la direction de rayonnement. 
  • Les champs en blanc peuvent être employés pour des annotations. Ces champs ne sont pas pris en compte dans le calcul et ne sont pas exprimés.

Entrez les données dans les champs

Lors de l’entrée des données, il faut procéder comme suit : 

  • Cliquez avec la souris sur le champ désiré et son fond deviendra bleu.
  • Entrez texte et données. Le fond redeviendra blanc.
  • Finir l’entrée du champ en pressant la touche "Enter". Le champ redeviendra bleu.
  • Choisir le champ suivant avec la souris ou les touches de direction.
  • Ne pas oublier ! Pressez la touche "Enter" après avoir fait la dernière entrée, sinon le contenu des champs ne sera pas enregistré.

Chargement des types d’antennes

Les types d’antennes peuvent être chargés depuis les dossiers suivants :

  • "Ant_Dat"
  • "Eigene"

Tous les types disponibles sont affichés. Les types d’antennes qui sont livrés avec le programme et se trouvant dans le dossier "Ant_Dat", sont protégés en écriture et ne se laissent pas modifier. Par contre, si on modifie dans le masque de données de l’antenne le "Type", celui-ci sera enregistré dans le dossier "Eigene" comme nouveau type.

Les entrées d’antennes personnelles peuvent être modifiées et seront enregistrées sous le même type avec "Enregistrer modifications". Si le "Type" dans le masque d’entrée est modifié, les entrées seront enregistrées avec "Nouveau type d’antenne" comme nouveau type.

Enregistrement des données d’antennes :

Les possibilités suivantes sont disponibles :

  • "Enregistrer comme nouveau type d’antenne" La désignation du type donnera automatiquement le nom du fichier qui sera ajouté dans la "Liste des types d’antennes".
  • "Enregistrer modifications" Les modifications effectuées aux types d’antennes du dossier "Eigene" seront enregistrées sur le même type. Les données seront modifiées.

 

Liste des types d’antennes


La liste des types d’antennes affiche tous les types présents dans le dossier "Eigene". Un type d’antenne peut être choisi avec la souris et sera affiché avec un clic sur le bouton "charger le type marqué".

 


Si un type d’antenne n’est plus d’actualité, il sera éliminé en cliquant sur le menu "Fichier" et dans sa fenêtre sur "effacer le type marqué".

L’entrée de type "1" Modèle ne peut pas être effacée.

 

 

Entrée des données de câbles


Le masque pour les données de câbles peut être ouvert en cliquant dans la fenêtre "Entrée des données du câble" sur "Masque pour l'entrée des données de câbles".

 

 

Masque pour les données de câbles


Entrer les données de câbles

L’entrée de données doit être faite de la manière suivante :

  • Cliquez sur le champ désiré. Le fond de celui-ci deviendra bleu.
  • Entrez texte et données. Le champ sera blanc.
  • Terminez l’entrée en pressant la touche "Enter". Le champ sera de nouveau bleu.
  • Sélectionnez le prochain champ avec la souris ou les touches de direction.
  • Ne pas oublier ! Pressez la touche "Enter" après avoir fait la dernière entrée pour que les valeurs soient enregistrée dans le masque.

Entrer les types de câbles

  • Marquez le champ dans la colonne respective.
  • A l’aide de la souris, modifiez le champ "Type de câble du fabricant".
  • Entrez le type du câble. Chaque caractère entré apparaît dans le champ vert de la colonne respective.
  • Retournez  dans la colonne désirée avec la souris.
